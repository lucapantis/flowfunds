# FlowFunds

FlowFunds is a personal finance tracker built with React, TypeScript and Tailwind CSS.

It helps users track income, expenses, available balance and recent transactions through a clean dashboard interface.

## Live Demo

https://flowfunds-two.vercel.app

## Features

- Financial dashboard with income, expenses and available balance
- Add new income or expense transactions
- Delete transactions
- Search transactions by description or category
- Filter transactions by type and category
- Shared transaction state between Dashboard and Transactions pages
- Responsive layout for desktop and mobile

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router
- Vite
- Vercel

## Project Status

The frontend is currently implemented with local React state and mock data. Data resets on page refresh because there is no backend or database yet.

Backend, authentication and database integration will be added later.
